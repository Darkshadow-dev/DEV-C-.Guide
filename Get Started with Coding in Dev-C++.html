About Dev-C++

Dev-C++ is an integrated development environment (IDE) for the C and C++ programming languages. It provides a user-friendly interface for writing, compiling, debugging, and executing code.

It uses the MinGW compiler and supports syntax highlighting, auto-completion, and project management.

About MinGW

MinGW (Minimalist GNU for Windows) is a minimalist development environment for native Microsoft Windows applications. It provides a collection of free and open-source tools, including the GCC (GNU Compiler Collection), which can compile C, C++, and other languages.

Example: Basic I/O


// C++ Basic I/O Example

Loops in C++

Type Description
For Loop Runs a block of code a fixed number of times.
While Loop Runs while a condition is true.
Do-While Loop Runs at least once, then checks condition.

Example of a For Loop


#include <iostream>
using namespace std;

int main() {
    for (int i = 0; i < 6; i++) {
        cout << "Loop iteration: " << i << endl;
    }
}

Functions in C++

Type Description
Regular Functions Reusable blocks of code defined once and called when needed.
Function Parameters Allow passing values into functions.
Return Values Functions can return a value after execution.

Example


// Function Prototype
int add(int a, int b);

// Function Definition
int add(int a, int b) {
    return a + b;
}

int main() {
    int result = add(5, 10);
    cout << "Result: " << result << endl;
}

Arrays in C++

Array Type Description
1D Array Stores a sequence of elements in a single line.
2D Array Represents data in rows and columns (like a matrix).
Multi-Dimensional Array Arrays with more than two dimensions.

Example


int numbers[5] = {1, 2, 3, 4, 5};
for (int i = 0; i < 5; i++) {
    cout << "Element " << i << ": " << numbers[i] << endl;
}

File Input/Output in C++

Operation Description
Reading from Files Use ifstream to read data from files.
Writing to Files Use ofstream to write data to files.
File Management Open, close, and manipulate files using fstream.

Example


#include <fstream>
#include <iostream>
using namespace std;

int main() {
    ofstream outFile("example.txt");
    outFile << "Hello, file!" << endl;
    outFile.close();

    ifstream inFile("example.txt");
    string line;
    while (getline(inFile, line)) {
        cout << line << endl;
    }
    inFile.close();
}

OOP Example


class Car {
public:
    string brand;
    int year;

    void displayInfo() {
        cout << brand << " - " << year << endl;
    }
};

int main() {
    Car myCar;
    myCar.brand = "Toyota";
    myCar.year = 2020;
    myCar.displayInfo();
}
