Welcome!

This page is your step-by-step C++ beginner guide. You’ll learn what int, bool, #include, loops, and more mean — with examples, explanations, and pictures.

🔹 Integers with int

int is used for whole numbers (no decimals). It’s one of the most common variable types in C++. 


// Example: using int
#include <iostream>
using namespace std;

int main() {
    int age = 18;
    cout << "My age is: " << age << endl;
    return 0;
}

👉 Here we created a variable age and gave it the value 18.

🔹 The #include Directive

#include tells the compiler to include extra functionality. For example, #include <iostream> gives access to input/output (cin, cout). 


#include <iostream> // allows input/output
using namespace std;

int main() {
    cout << "Hello, World!" << endl;
    return 0;
}

👉 Without #include <iostream>, cout and cin won’t work.

🔹 Loops

Loops repeat code automatically. C++ has 3 main types: for, while, and do...while. 


// Example: for loop
#include <iostream>
using namespace std;

int main() {
    for (int i = 1; i <= 5; i++) {
        cout << "Loop count: " << i << endl;
    }
    return 0;
}

👉 This will print numbers from 1 to 5. for = best when you know how many times to repeat. while = runs as long as a condition is true. do...while = runs at least once, then checks the condition.

🔹 Boolean Values (bool)

A bool stores only two values: true (1) or false (0). 


#include <iostream>
using namespace std;

int main() {
    bool isCodingFun = true;
    bool isHard = false;

    cout << "Is coding fun? " << isCodingFun << endl;
    cout << "Is it hard? " << isHard << endl;

    return 0;
}

👉 Output will be 1 for true and 0 for false.

🔹 The break Statement

break is used to stop a loop immediately, even if the condition isn’t finished. 


#include <iostream>
using namespace std;

int main() {
    for (int i = 1; i <= 10; i++) {
        if (i == 5) {
            break; // exit loop when i == 5
        }
        cout << i << endl;
    }
    return 0;
}

👉 The loop stops at 4 because break ends it when i == 5.

🔹 The goto Statement

goto jumps directly to a labeled part of the program. ⚠️ It should be avoided in real projects because it makes code hard to read. 


#include <iostream>
using namespace std;

int main() {
    int num = 1;

start:
    cout << "Number: " << num << endl;
    num++;
    if (num <= 5) goto start; // jump back to start

    return 0;
}

👉 This prints numbers 1–5 using goto. Better alternative: use loops instead.

🔹 Decisions with if / else

The if statement checks a condition. If it’s true → code runs. Otherwise → else runs. 


#include <iostream>
using namespace std;

int main() {
    int age = 16;

    if (age >= 18) {
        cout << "You are an adult." << endl;
    } else {
        cout << "You are a minor." << endl;
    }
    return 0;
}

👉 Since age is 16, the program prints You are a minor.

🔹 Multiple Choices with switch

switch checks one variable against many cases. It’s cleaner than writing many if statements. 


#include <iostream>
using namespace std;

int main() {
    int day = 3;

    switch(day) {
        case 1: cout << "Monday"; break;
        case 2: cout << "Tuesday"; break;
        case 3: cout << "Wednesday"; break;
        default: cout << "Other day";
    }
    return 0;
}

👉 Output: Wednesday. The break prevents running the next case.

🔹 Arrays (Lists of Values)

An array stores multiple values of the same type in one variable. Index starts from 0. 


#include <iostream>
using namespace std;

int main() {
    int numbers[5] = {10, 20, 30, 40, 50};

    cout << "First number: " << numbers[0] << endl;
    cout << "Third number: " << numbers[2] << endl;

    return 0;
}

👉 Output: 10 and 30.

🔹 Functions

Functions let you reuse code by giving it a name and calling it when needed. 


#include <iostream>
using namespace std;

void greet() {
    cout << "Hello, welcome to C++!" << endl;
}

int main() {
    greet(); // call the function
    return 0;
}

👉 Output: Hello, welcome to C++! Functions keep programs clean and organized.

🔹 Classes & Objects (OOP Basics)

A class is a blueprint, and objects are made from it. 


#include <iostream>
using namespace std;

class Car {
public:
    string brand;
    int year;

    void honk() {
        cout << "Beep beep!" << endl;
    }
};

int main() {
    Car myCar;
    myCar.brand = "Toyota";
    myCar.year = 2020;

    cout << myCar.brand << " (" << myCar.year << ")" << endl;
    myCar.honk();

    return 0;
}

👉 Output: Toyota (2020) and Beep beep!. Classes = objects + data + functions.

🔹 Dynamic Memory

With new, you can create variables or arrays at runtime. Always free them with delete. 


#include <iostream>
using namespace std;

int main() {
    int* p = new int; // allocate memory
    *p = 100;
    cout << "Dynamic value: " << *p << endl;

    delete p; // free memory
    return 0;
}

👉 Useful when you don’t know how much data you’ll need in advance.

🔹 Inheritance

A class can inherit from another class (reusing code). 


#include <iostream>
using namespace std;

class Animal {
public:
    void eat() {
        cout << "Eating..." << endl;
    }
};

class Dog : public Animal {
public:
    void bark() {
        cout << "Woof!" << endl;
    }
};

int main() {
    Dog myDog;
    myDog.eat();
    myDog.bark();
    return 0;
}

👉 Dog inherits from Animal, so it can both eat and bark.
